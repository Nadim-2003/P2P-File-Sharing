- Tab between fields
- Enter to submit
- Ctrl+A to select all
- Ctrl+C to copy

### Visual Feedback
- Button hover effects
- Highlighted selections
- Color-coded statuses
- Clear labels

### Text Sizes
- Small labels: 9pt
- Normal text: 10pt
- Emphasis: 11-12pt
- Headers: Bold, 10pt

---

## Window Sizing

### Recommended Size
- **Width**: 1200px (minimum 800px)
- **Height**: 900px (minimum 600px)
- **Aspect Ratio**: 4:3

### Scrollable Sections
- Download History
- Shared Files List
- Active Transfers
- Statistics Log
- Status Log
